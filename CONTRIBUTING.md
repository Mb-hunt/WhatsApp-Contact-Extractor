# Contributing to WhatsApp Group Contact Extractor

First off, thanks for considering contributing! 🎉 This project is simple by design, and every improvement helps thousands of marketers and community managers worldwide.

## How Can I Contribute?

### 🐛 Report Bugs

If the script doesn't work as expected:

1. Open an [Issue](https://github.com/Mb-hunt/WhatsApp-Contact-Extractor/issues)
2. Include:
   - Your browser (Chrome, Firefox, Edge, etc.) and version
   - What you expected to happen
   - What actually happened
   - Any error messages from the console (screenshot helps!)

### 💡 Suggest Features

Have an idea? Open an [Issue](https://github.com/Mb-hunt/WhatsApp-Contact-Extractor/issues) with the label `enhancement` and describe:

- What problem it solves
- How you imagine it working

### 🔧 Submit Code Changes

1. **Fork** this repository
2. **Create** a feature branch: `git checkout -b feature/my-improvement`
3. **Make** your changes to `whatsapp_extractor_v3.js`
4. **Test** your changes on WhatsApp Web with at least one group
5. **Commit**: `git commit -m 'Add my improvement'`
6. **Push**: `git push origin feature/my-improvement`
7. **Open** a Pull Request

### 📖 Improve Documentation

Found a typo? Instructions unclear? PRs for documentation improvements are always welcome.

## Guidelines

- Keep it simple — this is a single-file tool, let's keep it that way
- Test on multiple browsers if possible (Chrome, Firefox, Edge)
- Don't add external dependencies — the script must run standalone in a browser console
- Be respectful in all interactions

## Code Style

- Use clear variable names
- Add comments for non-obvious logic
- Keep console output user-friendly with emojis for status indicators

## Questions?

Open an issue — we're happy to help! 🙌
