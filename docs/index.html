How It Works

5 simple steps. No technical skills required.

1

Open WhatsApp Web

Go to web.whatsapp.com and connect your WhatsApp account.

2

Open the Members List

Open your group chat → click the group name → open the members panel.

3

Paste the Script

Press F12 → Console tab → paste the script from GitHub → hit Enter.

4

Download Your File

Wait 1-3 minutes. A .tsv file auto-downloads with all contacts. Open in Excel or Google Sheets.
